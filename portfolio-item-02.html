ClippyGPT

A React App

An AI hackathon project that I won for ASU Prep Digital to utilize AI and make a tool to help teachers grade writing assignements.

I used OpenAI's API to power the chatbot, I also used Firebase's Realtime DB to store data, and Vercel to host the application.

Technolgies Used: React, NextJS, Firebase
